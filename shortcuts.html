Computer Shortcuts Every Beginner Should Know

Shortcut Function
Ctrl + A Select Everything On the Page
Ctrl + V Paste
Ctrl + C Copy
Ctrl + Y Redo Last Action
Ctrl + S Save
Ctrl + X Cut
Ctrl + P Print
Ctrl + Shift + Esc Open Task Manager quickly
Ctrl + Shift + N Create A New Folder
Shift + Delete Permanently Delete File (skip recycle bin)
Alt + Tab Switch between Open Apps
Alt + Enter Open Properties of A Selected File
Alt + Space Open Window Control Menu
Windows + D Show/hide Desktop
Windows + E Open File Explorer
Windows + L Lock your Computer
Windows + I Open Settings
Windows + R Open Run Dialog
Windows + Tab Open Task View (see all desktops)
Alt + F4 Shut down / Restart / Sleep
